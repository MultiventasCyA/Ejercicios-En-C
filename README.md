# Ejercicios-En-C

Este repositorio contiene una colección de ejercicios prácticos desarrollados en lenguaje C, enfocados en lógica de programación y estructuras de datos.

## 🚀 Contenido
* **Calculadora:** Implementación básica de operaciones aritméticas.
* **Arreglos:** Ejercicios para calcular promedios y búsqueda de elementos.
* **Punteros:** Pruebas de manejo de memoria (en desarrollo).

## 🛠️ Requisitos
Para compilar y ejecutar estos programas, necesitas un compilador de C (como GCC).

## 💻 Cómo compilar y ejecutar
1. Abre la terminal.
2. Compila el archivo:
   `gcc nombre_del_archivo.c -o programa`
3. Ejecuta el programa:
   `./programa`
